Contact Us

Commercial Enquiries

For enquiries about commercial licensing, support, training, and consulting, please contact sales. Alternatively, if you already have a support contract established with Pivotal, please see the Pivotal support page.

If you have questions about how to use RabbitMQ and do not want to purchase a support contract or learn about commercial offerings, please use the RabbitMQ mailing list instead.

Community Support (Mailing List, IRC)

Public Mailing List

rabbitmq-users is the group for discussions about RabbitMQ. Once you join you can get help with technical issues, ask questions and share your experiences with RabbitMQ.

Slack

Feel free to join our community-driven RabbitMQ Slack channel.

IRC

RabbitMQ IRC channgel is #rabbitmq on irc.freenode.net: #rabbitmq -- irc://irc.freenode.net/rabbitmq. There's a Web interface to Freenode if you don't have an IRC client installed.

Private Enquiries

To contact the team privately, please email on info@rabbitmq.com. Please note that this should only be used for things that need to be discussed in private such as security vulnerabilities; if you want to ask a question about how to use RabbitMQ please use rabbitmq-users instead.
