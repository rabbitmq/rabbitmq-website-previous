Clients in other languages

The tutorials here use Python, Java, Ruby and PHP, however, code samples are available in a variety of languages:

  • C# (using .net/c# client)
  • clojure (using Langohr)
  • erlang (using erlang client)
  • java (using java client)
  • perl (using Net::RabbitFoot)
  • python (using pika)
  • python-puka (using puka)
  • ruby (using Bunny)
  • ruby (using amqp gem)

We also maintain a list of tutorials and a list of clients and developer tools for a range of platforms, which you may find useful.
