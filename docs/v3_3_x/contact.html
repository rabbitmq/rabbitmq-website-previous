Contact Us

Mailing List

rabbitmq-users is the list for discussions about RabbitMQ. Once you join you can get help with technical issues, ask questions and share your experiences with RabbitMQ. RabbitMQ team members are on this list.

Legacy Mailing List

rabbitmq-discuss is our legacy mailing list. It is hosted by the RabbitMQ team and is still active but will be discontinued eventually.

  • Read the list archive
  • You can also access the list through gmane, nabble

If you want to be notified of new releases and other important RabbitMQ announcements please sign up to the rabbitmq-announce list.

Commercial Enquiries

For enquiries about commercial licensing, support, training, and consulting, please contact us by email on support@rabbitmq.com. Alternatively, if you already have a support contract established with Pivotal, please see the Pivotal support page.

Social Media

Twitter

RabbitMQ is on twitter. If you're tweeting about RabbitMQ please don't forget to use the #rabbitmq hash tag.

IRC

RabbitMQ developers are sometimes available to chat rabbitmq on irc.freenode.net (see also http://freenode.net/) in channel #rabbitmq -- irc://irc.freenode.net/rabbitmq. The logs of these conversations are available at http://dev.rabbitmq.com/irclog/. You can use this web interface to freenode if you don't have an IRC client installed.

Facebook

RabbitMQ is on Facebook.
